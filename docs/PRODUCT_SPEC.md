# Product spec: Календарь накоплений

## Цель

Помочь пользователю откладывать небольшую сумму денег по выбранной схеме и видеть накопленный прогресс.

## MVP

Пользователь может:

- увидеть сумму на сегодня;
- подтвердить, что отложил деньги;
- увидеть баланс по подтверждённым дням;
- увидеть прогноз до конца выбранного периода;
- посмотреть календарь текущего месяца;
- посмотреть историю подтверждённых взносов;
- изменить базовую ставку;
- выбрать первый и последний день периода накоплений;
- отмечать будущие дни внутри периода накоплений;
- переключиться с нарастающей суммы на ровную сумму;
- открыть помощь с инструкцией по использованию приложения;
- включить/выключить ежедневные напоминания;
- включить/выключить ежемесячные отчёты.

## Правила накоплений

По умолчанию период накоплений длится с 1 января по 31 декабря текущего года.

Режим суммы по умолчанию:

```text
amount = dayNumberInPeriod * baseRate
```

`dayNumberInPeriod` считается от выбранной даты начала накоплений.

Режим ровных сумм:

```text
amount = baseRate
```

Изменение ставки, периода или режима не пересчитывает уже подтверждённые записи.

## Ежемесячный отчёт

Каждый последний день месяца приложение показывает ободряющий отчёт:

- сколько отложено за дни текущего месяца внутри выбранного периода;
- сколько отложено за весь выбранный период;
- сколько дней периода в этом месяце отмечено;
- процент выполненных дней периода в месяце.

## Не входит в MVP

- Банковские переводы.
- Регистрация.
- Облачная синхронизация.
- Несколько копилок.
- Сложные графики.
- Экспорт.
- Виджеты.
