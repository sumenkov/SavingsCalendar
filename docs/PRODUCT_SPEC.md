# Product spec: Календарь накоплений

## Цель

Помочь пользователю ежедневно откладывать небольшую сумму денег по простой нарастающей схеме и видеть накопленный прогресс.

## MVP

Пользователь может:

- увидеть сумму на сегодня;
- подтвердить, что отложил деньги;
- увидеть баланс по подтверждённым дням;
- увидеть прогноз до конца года;
- посмотреть календарь текущего месяца;
- посмотреть историю подтверждённых взносов;
- изменить базовую ставку;
- включить/выключить ежедневные напоминания;
- включить/выключить ежемесячные отчёты.

## Ежемесячный отчёт

Каждый последний день месяца приложение показывает ободряющий отчёт:

- сколько отложено за текущий месяц;
- сколько отложено с начала года;
- сколько дней месяца отмечено;
- процент выполненных дней.

## Не входит в MVP

- Банковские переводы.
- Регистрация.
- Облачная синхронизация.
- Несколько копилок.
- Сложные графики.
- Экспорт.
- Виджеты.
