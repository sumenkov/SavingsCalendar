# Календарь накоплений

**Календарь накоплений** - простое Android-приложение для ежедневной привычки откладывать деньги.

Идея такая: каждый день приложение показывает, сколько нужно отложить, а вы отмечаете выполненный взнос в календаре. Так маленькие ежедневные действия превращаются в понятный план накоплений, без таблиц, блокнотов и ручных подсчётов.

## Для чего это приложение

Приложение помогает:

- видеть сумму взноса на сегодня;
- отмечать выполненные дни;
- смотреть баланс выбранного периода;
- смотреть историю всех сделанных взносов;
- понимать прогноз до конца периода накоплений;
- не забывать о взносе благодаря ежедневным напоминаниям;
- получать итоговый отчёт в последний день месяца;
- получать предложение скачать и установить новую версию из GitHub Releases.

Приложение не переводит деньги автоматически и не подключается к банкам. Это личный календарь привычки: вы откладываете деньги удобным способом, а приложение помогает вести учёт.

## Как работает схема накоплений

По умолчанию используется нарастающая схема:

```text
сумма дня = номер дня периода × базовая ставка
```

Если базовая ставка равна `1 ₽`, то:

```text
1-й день = 1 ₽
2-й день = 2 ₽
3-й день = 3 ₽
...
53-й день = 53 ₽
```

Базовую ставку можно изменить в настройках. Например, если поставить `10 ₽`, то первый день будет `10 ₽`, второй `20 ₽`, третий `30 ₽`.

Также есть режим **ровных сумм**. В нём каждый день откладывается одна и та же сумма.

## Установка

### Вариант 1. Скачать APK с GitHub вручную

1. Откройте страницу релизов: <https://github.com/sumenkov/SavingsCalendar/releases/latest>.
2. В блоке **Assets** скачайте APK-файл вида `savings-calendar-<version>.apk`.
3. Откройте скачанный APK на Android-устройстве.
4. Если Android попросит разрешить установку из этого источника, разрешите её.
5. Завершите установку и откройте **Календарь накоплений**.

Если в релизе нет APK-файла в **Assets**, значит релиз собран неправильно и приложение не сможет обновиться автоматически.

### Вариант 2. Установить APK из файла

1. Скачайте APK-файл приложения.
2. Откройте файл на Android-устройстве.
3. Если система попросит разрешить установку из этого источника, разрешите её.
4. Завершите установку и откройте **Календарь накоплений**.

### Требования

- Android 8.0 или новее.
- Для напоминаний нужно разрешение на уведомления.
- На некоторых версиях Android для точного времени напоминаний может понадобиться разрешение на точные будильники.

## Первый запуск

После установки откройте приложение и проверьте настройки:

1. Откройте вкладку **Настройки**.
2. Укажите базовую ставку или включите режим ровных сумм.
3. Проверьте символ валюты. По умолчанию используется `₽`.
4. Выберите период накоплений. По умолчанию это годовой период с 1 января по 31 декабря.
5. Включите ежедневные напоминания, если хотите получать уведомление каждый день.
6. Включите месячные отчёты, если хотите получать итог в последний день месяца.

После этого можно вернуться на вкладку **Сегодня** и отметить первый взнос.

## Основные вкладки

### Сегодня

Главный экран приложения. Здесь видно:

- текущую дату;
- номер дня в выбранном периоде накоплений;
- сумму, которую нужно отложить сегодня;
- баланс выбранного периода;
- прогноз до конца периода;
- краткий месячный отчёт.

Если сегодня не входит в выбранный период, приложение покажет, что период ещё не начался или уже завершён.

Нажмите **Сделать взнос**, когда отложили деньги за сегодня.

### Календарь

На вкладке **Календарь** можно смотреть дни месяца и вручную отмечать взносы.

Цвета дней помогают быстро понять состояние:

- отмеченный день уже добавлен в накопления;
- сегодняшний день выделен отдельно;
- прошлые пропущенные дни заметны в календаре;
- дни вне периода накоплений нельзя отметить.

Выберите день, чтобы увидеть подробности: сумму, ставку, номер дня периода и статус. Уже отмеченный день можно отменить.

### История

На вкладке **История** отображаются все подтверждённые взносы, сгруппированные по месяцам.

Здесь можно:

- посмотреть общий итог по всей истории;
- увидеть сумму за каждый месяц;
- проверить отдельные взносы;
- отменить ошибочно отмеченный день.

### Настройки

В настройках можно изменить поведение приложения:

- базовую ставку;
- символ валюты;
- режим расчёта: нарастающая сумма или ровная сумма;
- начало и конец периода накоплений;
- ежедневные напоминания;
- время ежедневного напоминания;
- месячные отчёты;
- время месячного отчёта;
- возможность отмечать прошлые дни.

Также в настройках есть встроенная помощь с объяснением расчётов, отметок, баланса и прогноза.

## Напоминания и месячные отчёты

Приложение умеет отправлять два типа уведомлений:

1. **Ежедневное напоминание** - помогает не забыть сделать взнос.
2. **Месячный отчёт** - приходит в последний день месяца и показывает:
    - сколько отложено за дни текущего месяца внутри выбранного периода;
    - сколько отложено за весь выбранный период;
    - сколько дней периода в этом месяце отмечено;
    - процент выполненных дней периода в месяце.

Если уведомления не приходят, проверьте разрешения Android для приложения.

## Обновления

При запуске приложение проверяет последний публичный GitHub Release. Если приложение было свернуто минимум на 15 минут, проверка повторяется при возврате. Пока приложение открыто, оно также проверяет обновления по таймеру примерно раз в час.

Если там есть новая версия и APK-файл в **Assets** релиза, приложение предложит скачать APK и открыть системный установщик Android.

Для установки может понадобиться разрешить установку приложений из этого источника в настройках Android.

Обновление можно скачать и вручную: откройте <https://github.com/sumenkov/SavingsCalendar/releases/latest>, скачайте APK из **Assets** и запустите его на устройстве.

## Важные правила расчёта

- Баланс считается только по подтверждённым взносам.
- Баланс периода считает только взносы внутри выбранного периода.
- История показывает все подтверждённые взносы, даже если они вне текущего периода.
- Пропущенные прошлые дни не добавляются в баланс автоматически.
- Прогноз считает уже подтверждённый баланс плюс план от сегодняшнего дня до конца периода.
- Изменение ставки, периода или режима влияет на новые расчёты.
- Уже подтверждённые взносы сохраняют свою фактическую сумму и не пересчитываются задним числом.

Это сделано специально, чтобы история оставалась честной: что было отмечено, то и хранится.

## Пример использования

Допустим, вы хотите копить весь год по схеме `1 ₽ × номер дня`.

1. Оставьте базовую ставку `1 ₽`.
2. Оставьте период с 1 января по 31 декабря.
3. Каждый день открывайте приложение.
4. Откладывайте указанную сумму.
5. Нажимайте **Сделать взнос**.
6. В конце месяца смотрите отчёт и радуйтесь, что копилка подросла.

Если схема становится слишком тяжёлой, можно уменьшить ставку, поменять период или перейти на ровную ежедневную сумму.

## Данные и приватность

Приложение хранит данные локально на устройстве:

- отмеченные взносы;
- выбранные настройки;
- время уведомлений;
- период накоплений.

Приложение не требует регистрации, не подключается к банку и не отправляет пользовательские данные на сервер. Для проверки обновлений приложение обращается к GitHub Releases.

## Частые вопросы

### Можно ли отмечать будущие дни?

Да. В календаре можно выбрать будущий день внутри периода накоплений и отметить его вручную.

### Можно ли отмечать прошлые дни?

Да, если включена настройка **Разрешить отметки прошлых дней**.

### Что будет, если изменить базовую ставку?

Новая ставка будет использоваться для новых отметок. Уже сохранённые взносы не изменятся.

### Что будет, если отменить взнос?

День снова станет неотмеченным, а сумма будет убрана из баланса и истории.

### Почему прогноз не догоняет пропущенные дни?

Потому что приложение не считает пропущенные взносы автоматически выполненными. Прогноз показывает реалистичный путь от текущего состояния до конца периода.

### Можно ли использовать другую валюту?

Да. В настройках можно изменить символ валюты, например на `₽`, `$`, `€` или другой короткий знак.

## Если что-то пошло не так

### Уведомления не приходят

Проверьте:

1. Разрешены ли уведомления для приложения в настройках Android.
2. Включены ли ежедневные напоминания в настройках приложения.
3. Не ограничивает ли система фоновую работу приложения.
4. Разрешены ли точные будильники, если Android показывает такое предупреждение.

### Нельзя отметить день

Возможные причины:

- день находится вне выбранного периода накоплений;
- это прошлый день, а отметки прошлых дней выключены;
- день уже был отмечен ранее.

### Сумма кажется неправильной

Проверьте:

- выбранный режим расчёта;
- базовую ставку;
- период накоплений;
- номер дня внутри периода.

## Коротко

**Календарь накоплений** - это маленькая копилка-дисциплинатор. Она не забирает деньги сама, не спорит с банками и не строит финансовую ракету. Она просто каждый день говорит: «сегодня нужно отложить вот столько», а потом аккуратно считает ваш прогресс.
